• terminology:
  • flexbox is about the relationship between a parent container and its direct descendant child items
  • parent container: flex container
  • direct descendants: flex items
  • flex items flow along the main axis, by default is left to right
  • main start: beginning of flex container, where flex items begin flow from
  • main end: opposing side of flex container
  • main size: length from main start -> main end
  • cross axis: runs perpendicular to the main axis
  • cross axis has cross start, cross end, cross size
