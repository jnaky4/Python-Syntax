• terminology:
  • grid is about the relationship between a parent container and its direct descendant child items
  • parent container: grid container
  • direct descendants: grid items
  • grid is made up of 2 sets of grid lines,
  • grid column: run along the column axis
  • grid row: run along the row axis
  • grid tracks: generic term of space between grid rows or grid columns
  • grid cells: the boxes between the grid lines
  • grid areas: any portion of the grid contained by four grid lines
